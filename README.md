# Excel Report Generator

**Excel Report Generator** — веб-приложение для автоматизированного анализа Excel-файлов и генерации аналитических PDF-отчетов.

Приложение позволяет загрузить таблицу в формате `.xlsx`, выполнить проверку файла, рассчитать основные показатели, построить графики, сформировать текстовые аналитические выводы и скачать готовый PDF-отчет.

---

## Возможности приложения

- загрузка Excel-файлов формата `.xlsx`;
- проверка корректности загружаемого файла;
- обработка ошибок при чтении Excel-документа;
- анализ табличных данных;
- расчет основных статистических показателей;
- построение графиков по данным из таблицы;
- автоматическое формирование текстовых аналитических выводов;
- генерация PDF-отчета;
- просмотр истории сформированных отчетов;
- скачивание отчетов из истории;
- удаление отчетов из истории.

## Основной сценарий работы

1. Пользователь загружает Excel-файл.
2. Система проверяет формат и структуру файла.
3. Приложение считывает данные из таблицы.
4. Выполняется расчет статистических показателей.
5. Формируются графики и аналитические выводы.
6. Пользователь просматривает результат на странице предпросмотра.
7. Система генерирует PDF-отчет.
8. Отчет сохраняется в истории и доступен для скачивания.

## Установка и запуск проекта

### 1. Создать виртуальное окружение

```bash
python -m venv .venv
```

### 2. Активировать виртуальное окружение

Для Windows PowerShell:

```bash
.venv\Scripts\activate
```

### 3. Установить зависимости

```bash
pip install -r requirements.txt
```

### 4. Запустить приложение

```bash
uvicorn app.main:app --reload
```

### 5. Открыть приложение в браузере

```text
http://127.0.0.1:8000
```

## Валидация файлов

Приложение выполняет проверку загружаемых файлов и выводит понятные сообщения об ошибках.

Проверяется:

- выбран ли файл;
- имеет ли файл расширение `.xlsx`;
- не является ли файл пустым;
- не превышает ли файл допустимый размер;
- содержит ли Excel-файл строки и столбцы;
- есть ли в таблице числовые столбцы для анализа;
- удалось ли корректно прочитать файл.

---

## Генерация PDF-отчета

PDF-отчет содержит:

- дату формирования отчета;
- общую информацию о таблице;
- основные числовые показатели;
- автоматические аналитические выводы;
- графики;
- предпросмотр первых строк исходной таблицы.

После формирования отчет сохраняется в папке `reports` и отображается на странице истории отчетов.

---

## История отчетов

В приложении реализована отдельная страница истории сформированных PDF-отчетов.

На этой странице можно:

- посмотреть список созданных отчетов;
- увидеть дату создания отчета;
- увидеть размер PDF-файла;
- скачать отчет;
- удалить ненужный отчет.
